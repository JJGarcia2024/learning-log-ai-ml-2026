# Forex & Metals Trading System Documentation

This document provides a comprehensive overview of the scripts, folders, and resources present in this branch (`learning-log-ai-ml-2026`). The primary function of this repository is to systematically analyze foreign exchange and precious metal pairs, applying strict technical indicators and macroeconomic fundamental scraping to output daily Markdown trade reports and position-sizing recommendations.

## 🐍 Python Scripts

### 1. `forex_report_generator.py`
The core engine for the forex market.
- **Purpose:** Analyzes 42 different forex pairs to generate standardized trading reports.
- **Data Acquisition:** 
  - Price Action: Uses `yfinance` to fetch 1-year historical price data at a daily interval.
  - Fundamentals: Features a custom web-scraper utilizing `urllib` and `xml.etree.ElementTree` to query Google News RSS feeds for macroeconomic and central bank news within the last 30 days, specific to the base and quote currencies.
- **Technical Analysis:** Calculates 9, 21, 50, and 200 Exponential Moving Averages (EMA) to determine the trend regime (e.g., UPTREND, DOWNTREND, CHOP). Also calculates RSI (14) and Average True Range (ATR) for volatility assessment.
- **Risk Management Engine:** Mathematically limits risk to 1.0% of a predefined account balance ($10.0 default). Dynamically calculates the stop loss distance (1.5 × ATR) and dynamically determines the exact **XM Micro Lot Size** allowable.
- **Outputs:** 
  - Individual Markdown files for all 42 pairs (e.g., `reports/Report_AUDCAD.md`).
  - An overarching master summary document (`reports/Tradeable_Forex_Universe.md`).

### 2. `metals_report_generator.py`
The dedicated engine for precious metals.
- **Purpose:** Analyzes 15 synthetic precious metal cross-pairs (e.g., Gold against EUR, JPY, AUD). 
- **Data Acquisition:** 
  - Combines spot FX prices with metal futures (e.g., `GC=F` for Gold, `SI=F` for Silver) to create "synthetic" pricing data for combinations that Yahoo Finance does not support natively.
  - Implements the exact same macroeconomic Google News RSS scraping system as the forex generator.
- **Technical Analysis & Risk:** Employs the identical technical architecture (EMAs, RSI, ATR) and risk management logic (1% risk on a $10 account).
- **Outputs:**
  - Individual Markdown files for all 15 metal pairs (e.g., `reports/Report_XAUEUR.md`).
  - An overarching master summary document (`reports/Tradeable_Metals_Universe.md`).
